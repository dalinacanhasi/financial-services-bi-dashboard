# Financial Services Business Intelligence Dashboard (Power BI)

## Author
Dalina Canhasi  
Business Intelligence & Data Analytics Portfolio Project

---

## Overview
This project demonstrates an end-to-end Business Intelligence solution built in Power BI using a financial services / structured products use case.

The dashboard focuses on two key areas:
1. Financial performance tracking (revenue, profit, growth)
2. Operational efficiency monitoring (SLA, automation, processing time)

The goal is to showcase practical BI skills including:
- Data modeling (star schema)
- DAX measures and time intelligence
- KPI design
- Interactive dashboards
- Business-focused analytics

---

## Tools Used
- Power BI Desktop
- DAX
- Excel (data source)
- Basic data modeling (fact + dimensions)

---

## Data Model
Star schema design:

Fact Table:
- Transactions (monthly client/product activity)

Dimensions:
- Date
- Client
- Product
- Country

This structure enables efficient filtering and scalable analytics.

---

## Key Metrics Implemented

### Financial KPIs
- Revenue
- Cost
- Gross Profit
- Gross Margin
- Month-over-Month Growth

### Operational KPIs
- SLA Compliance
- Automation Rate
- Average Processing Time

### Client Analytics
- Top 10 Clients by Revenue
- Client Revenue Concentration %

---

## Dashboard Pages

### Executive Overview
Tracks financial performance across business lines and geographies:
- Revenue trend by month
- Business line comparison
- Revenue by country
- Top clients ranking
- Revenue concentration risk

### Operations Performance
Monitors service delivery efficiency:
- SLA trends
- Automation vs processing time analysis
- Operational KPIs
- Regional comparison
